🎯 ШВИДКИЙ СТАРТ (3 КРОКИ):

   КРОК 1: Генерація SESSION (ЛОКАЛЬНО)
   ─────────────────────────────────────
   ./setup_render_session.sh
   
   Або вручну:
   python3 generate_string_session.py
   
   👉 СКОПІЮЙТЕ отриманий STRING_SESSION!

   КРОК 2: Додати в Render Dashboard
   ──────────────────────────────────
   1. Відкрийте https://dashboard.render.com
   2. Ваш сервіс (neptun) → Environment
   3. Add Environment Variable:
   
      Name:  TELEGRAM_SESSION
      Value: [вставте ваш STRING_SESSION]
   
   4. Також додайте (якщо немає):
      
      Name:  TELEGRAM_API_ID
      Value: 24031340
      
      Name:  TELEGRAM_API_HASH
      Value: 2daaa58652e315ce52adb1090313d36a
      
      Name:  SOURCE_CHANNELS
      Value: UkraineAlarmSignal,kpszsu,war_monitor,napramok,raketa_trevoga,ukrainsiypposhnik
      
      Name:  TARGET_CHANNEL
      Value: mapstransler
   
   5. Save Changes

   КРОК 3: Створити Background Worker
   ───────────────────────────────────
   1. Render Dashboard → Background Workers
   2. Add Background Worker:
   
      Name: channel-forwarder
      Start Command: python channel_forwarder_render.py
   
   3. Create Background Worker
   4. Manual Deploy
